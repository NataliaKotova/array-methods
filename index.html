Array methods: join, split, toString

join

w3schools The join() method returns the array as a string. The elements will be separated by a specified separator. The default separator is comma (,). Note: this method will not change the original array.
var fruits = ["Banana", "Orange", "Apple", "Mango"];

toString

w3schools The toString() method returns a string with all the array values, separated by commas. Note: This method will not change the original array.
var fruits = ["Banana", "Orange", "Apple", "Mango"];

split

w3schools The split() method is used to split a string into an array of substrings, and returns the new array. Tip: If an empty string ("") is used as the separator, the string is split between each character. Note: The split() method does not change the original string.
var string = "How are you doing today?";
